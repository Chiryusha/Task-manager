# React To Do List

Простое приложение списка задач, созданное на React. Позволяет добавлять, отмечать выполненными и удалять задачи.

## Функциональность

- Добавление новых задач
- Отметка задач как выполненных
- Удаление задач
- Удобный и простой интерфейс
- Работа в браузере без серверной части
- Редактирование задач
- Фильтрация задач

## Технологии

- React
- JavaScript
- HTML
- CSS
